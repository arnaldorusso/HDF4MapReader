aggregate_multidimensional_chunks(self, chunks, sds_info, current_chunk_offset=0, current_index=0, aggregated_sds_array=None)

 source code 

recursively aggregates multidimensional chunks to a multidimensional cube, the dimension must be the same for both.

Parameters:
  • chunks - instance of 'chunk' transforms plain bytes into a n-dimensional chunk.
  • sds_info - an instance of sds_info that holds information about the SDS.
  • current_chunk_offset - it is used to change the axis of aggregation as we go to a higher dimension.
  • current_index - index of the chunk we are aggregating.
  • aggregated_sds_array - final aggregated array.
